# 简播(SimpleLive)

## 项目背景

随着直播越来越普及，有时有一些突发奇想随时想要直播的需求，如朋友之间的直播聊天、直播会议等等。本项目旨在轻量级的Web直播平台，支持主播Web开播，观众Web看直播。用户随时浏览器开播，亦可下载第三方推流软件进行推流直播。



## 项目目标

- 实现直播流在浏览器中播放
- 支持直播列表与直播间页面
- 实现基础互动功能(弹幕/聊天/连麦)
- 实现基本的用户登录注册功能





## 用户角色

- 游客用户：浏览直播内容
- 登录用户：观看直播、参与互动
- 主播：开直播、禁言等等



## 功能需求

### 1、用户系统

- 登录/注册

### 2、直播首页

- 展示当前可观看的直播
- 显示直播标题、封面、主播头像、主播名、直播间人数
- 顶部导航栏：首页、直播、分类、搜索框、开播、关注、历史、用户个人简介

### 3、个人中心

- 我的信息：基础信息以及修改
- 关注与历史：关注与观看记录

### 4、直播播放页

- 播放直播流
- 显示直播信息

### 5、实时互动

- 聊天功能
- 弹幕功能
- 连麦功能
